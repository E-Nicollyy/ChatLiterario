# Chat Literário Inteligente

Aplicação desenvolvida em Python utilizando:

- Google Gemini
- Gradio
- Python-Docx

## Funcionalidades

- Indicações de livros
- Criação de textos literários
- Interpretação textual
- Correção e revisão de textos

## Tecnologias

- Python
- Gradio
- Gemini API
- Pillow
- Python-Docx
